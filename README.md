# 🚀 ResearchGPT

AI-Powered Research Intelligence Platform built using Gemini AI, Tavily Search, Streamlit, and Python.

## Features

✅ Web Research

✅ AI-Powered Analysis

✅ Executive Research Reports

✅ Statistics Extraction Agent

✅ Validation Agent

✅ PDF Report Export

✅ Markdown Export

✅ Research History

✅ Modern Interactive Dashboard

---

## Tech Stack

* Python
* Streamlit
* Gemini 2.5 Flash
* Tavily Search API
* ReportLab
* Git & GitHub

## Architecture

User Query
